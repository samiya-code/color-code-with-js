# Color Code Project

A simple web project demonstrating HTML, CSS, and JavaScript fundamentals with colored box and grid layouts.

## Project Overview

This project is a basic web development exercise that showcases:
- HTML structure with semantic elements
- CSS styling with colors, padding, margins, and grid layouts
- JavaScript DOM manipulation and basic programming concepts

## Files Structure

```
color code/
├── index.html      # Main HTML file with page structure
├── index.css       # CSS styling for colors and grid layout
├── index.js        # JavaScript code with commented examples
├── index.json      # VS Code settings (format on save, live server port)
└── README.md       # This documentation file
```

## Features

### HTML Structure
- Semantic HTML5 structure
- A main content box with heading
- Grid layout with three colored sections

### CSS Styling
- **Box Component**: Red background with padding and margins
- **Grid Layout**: 
  - White background container
  - Three colored sections (brown, purple, blueviolet)
  - Grid template columns and rows configuration
  - Inline display for grid items

### JavaScript
- Contains commented code examples demonstrating:
  - String concatenation
  - Conditional statements
  - Function declarations and expressions
  - Arrow functions
  - DOM manipulation (attempted getElementById usage)

## Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, etc.)
- A code editor (VS Code recommended)
- Live server extension for VS Code (optional but recommended)

### Installation
1. Clone or download this project
2. Open the project folder in your code editor
3. If using VS Code, install the Live Server extension
4. Right-click on `index.html` and select "Open with Live Server"

### Manual Setup
If not using Live Server:
1. Open `index.html` directly in your web browser
2. The page will load with the styled elements
